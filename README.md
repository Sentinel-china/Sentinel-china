# SENTINEL - 工业自动化解决方案

## 项目简介

SENTINEL是一家专注于工业自动化解决方案的公司，提供从传感器到连接系统的完整产品线。我们致力于通过创新技术帮助客户实现工厂自动化转型和升级，提高业务效率和竞争优势。

## 核心产品

### 🔍 传感器 (Sensors)
- 温度传感器 (Temperature Sensor)
- 压力传感器 (Pressure Sensor)  
- 热流传感器 (Thermal Flow Sensor)
- 涡流传感器 (Vortex Flow Sensor)

### 🔌 I/O 模块 (I/O Modules)
- PROFINET 协议支持
- CC-Link IE Field Basic
- EtherCAT 协议
- 自适应输入/输出

### 📡 IO-Link 技术
- IO-Link Master
- IO-Link Hubs
- IO-Link Converter
- 传感器与PLC通信

### 🔗 连接系统 (Connectivity)
- 7/8" 电源电缆
- 接线盒 (Junction Boxes)
- 工业以太网电缆
- 连接器套件 (Cordsets)

## 服务流程

1. **需求分析** - 深入了解客户需求，制定详细产品方案
2. **选择定制** - 匹配产品并定制解决方案
3. **交付** - 按时发货
4. **售后服务** - 提供优质的售后服务

## 技术栈

- **前端框架**: React + TypeScript
- **样式框架**: Tailwind CSS
- **图标库**: Lucide React ：https://lucide.dev/icons/
- **路由**: React Router
- **构建工具**: Vite
